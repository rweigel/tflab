Overview

Comparison of two approaches to determining transfer function for an infinite half-plane.

Relationship between Ex and By or dBy/dt for infinite half-plane:

Ex(t) = sum_w cos(w*t)
By(t) = sum_w w^(-3/2)*cos(w*t - pi/4 + phi_w)
dBy/dt = sum_w w^(-1/2)*cos(w*t - pi/4 + phi_w)

Method

Create time series of each for w = 2*pi*f = [1:N/2]/N, with N large. To make both time series stationary, add a random phase phi_w for each w. (Time domain approach requires stationarity.)

Frequency domain approach

Transfer function is ratio of By(f)/Ex(f).

Time domain approach

Impulse response coefficients are estimated. Transfer function is periodogram of impulse response coefficients. 800 coefficients are used (400 causal, 400 acausal).
