The general problem is that given a scalar output signal $y(t)$ and an input signal $\mathbf{x}(t)$ with $N_{in}$ components, compute the complex--valued impedance $Z(f)$

$Y(f) = \mathbf{Z}(f)\cdot\mathbf{X}(f)$

where $\mathbf{Z}$ and $\mathbf{X}$ are matrices with $N_f\times N_{in}$ elements.

We assume that the user has measurements that have have underwent visual quality control and that artifacts such as spikes and baseline offset changes have been removed.
We also assume that there are one or more intervals of $N_t$ measurements of $\mathbf{x}$ and $\mathbf{y}$ on a uniform time grid.

To illustrate ...

If the Fourier transform of the input, $\mathbf{X}(f)$ has one component, then

$Y(f) = Z(f)X(f)$

A simple estimate of $Z$ can be obtained by computing the $N_t$--point DFT of $y(t)$ and $\mathbf{x}(t)$ and estimate $Z$ using

$\hat{Z}(f_k) = \frac{Y(f_k)}{X(f_k)}$

where $f_k$ are the DFT frequencies: $f_k = k/N_t$ for $k = 0, 1,..., N_t-1$.

If $x(t)$ has the form 

$x(t)=\sum_{k=0}^N_t \cos(2\pi f_k t)+\sin(2\pi f_k t)$

and $y(t)$ has the form

$y(t)=\sum_{k=0}^N_t Z_{rk}\cos(2\pi f_k t)+Z_{ik}\sin(2\pi f_k t)$

one will find that $\hat{Z}(f_k)=Z(f_k)$ to numerical precision for all $\hat{Z}(f_k)\ne 0$. For $f_k$ where $Z(f_k)=0$, $\hat{Z}(f_k)$ will not be zero to numerical precision and their value depends on the algorithm used to compute the DFT.

$\hat{Z}(f_k) = \frac{Y(f_k)}{X(f_k)}$

$\hat{Z}(f_k) = \frac{\sum_{t=0}^{N_t-1} y(t)e^{-2\pi j f_k (t-1)/N_t}}{\sum_{t=0}^{N_t-1} x(t)e^{-2\pi j f_k (t-1)/N_t}},

This constraint is unlikely to be satified -- typically there is (1) measurement error associated with both $y(t)$ and $\mathbf{x}(t)$ and the signals available for measurement are (2) generated by processes that have a contiuous spectrum so that a time interval of $y(t)$ cannot be selected that is exactly periodic.
